# Contract Insight (MVP)

Full-stack AI contract analysis tool.

## Tech Stack
- Backend: Spring Boot (Java)
- Frontend: ReactJS
- AI: Hybrid LLM (Cloud API + Local processing)
- Deployment: Docker (VPS)

## Features
- Upload contract (PDF/Text/Image)
- Extract text from document
- AI summary of contract
- Key clause extraction
- Risk detection
- Export result

## Purpose
MVP project to demonstrate full-stack development + AI integration (hybrid cloud/local LLM).
